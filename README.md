## Init Loacl ENV

### install pipenv & virtualenv
```
pip install --user pipenv
pip install virtualenv
```

### init virtualenv 
```shell script
cd /project/dir
virtualenv venv
source ./venv/bin/activate
```

### install requirements
```shell script
pip install -r requirements.txt
```
